# Pricing Rules (Mandatory)

## INR Products

If Original Price < ₹1,000

Final Price = ((Original Price + 2000) × 1.25)

If Original Price ≥ ₹1,000

Final Price = ((Original Price + 2000) × 1.22)

---

## Foreign Currency Products

Fetch the latest live exchange rate directly to INR.

If Original Price < 1000

Final Price = ((Original Price × (Today's INR Exchange Rate + 12)) + 2000) × 1.25

If Original Price ≥ 1000

Final Price = ((Original Price × (Today's INR Exchange Rate + 12)) + 2000) × 1.22

Rules:

* Use exactly one exchange rate lookup.
* Convert directly to INR.
* Never convert through another currency.
* Never convert twice.
* Never use cached or fallback exchange rates.
* Round only the final INR value.

# 2. Variant Availability Bug

Products are importing variants that are unavailable.

Example:

Website

* XL = Out of Stock

Shopify

* XL = Available
* Inventory = 100

This is incorrect.

Rules:

* Import only variants that are actually available.
* If a variant is sold out:

  * Skip the variant completely, **or**
  * Import it with Inventory = 0 and Shopify showing it as Sold Out.
* Never assign default inventory (100) to unavailable variants.
* Never guess stock availability.
